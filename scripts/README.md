# Scripts

Root-level development and CI scripts.

| Script | Description |
|--------|-------------|
| `merge-coverage.js` | Merges frontend and backend coverage reports (used by `npm run test:coverage`) |
| `start-ngrok.ps1` | Start ngrok for local tunnel (OAuth callbacks, etc.) |
| `test-email-endpoint.sh` | Test email delivery endpoint |
| `test-health.ps1` | Health check script for API |
